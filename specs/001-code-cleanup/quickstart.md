# Quickstart: Проверка очистки кода

**Feature**: 001-code-cleanup

## Как проверить изменения

1. Открыть `page1.html` в браузере
2. Проверить конфигурацию 2 подушки / 1 одеяло:
   - Нажать кнопку "2" у подушек, "1" у одеял
   - Нажать "ДАЛЕЕ"
   - Нажать на плюс на любой подушке
   - Выбрать размер → должны появиться цвета
   - Выбрать цвет → изображение должно смениться, должно появиться уведомление внизу справа
   - Нажать "ПОСМОТРЕТЬ СОСТАВ ЗАКАЗА"
   - Проверить что товар отображается в заказе с ценой
   - Нажать "×" для удаления товара
   - Нажать "НАЗАД"
3. Повторить для конфигурации 4 подушки / 2 одеяла
4. Открыть консоль браузера (F12):
   - Не должно быть потока отладочных сообщений
   - `console.error` для реальных ошибок должен работать
5. В консоли проверить уникальность ID:
   ```js
   // Должен вернуть пустой массив (нет дубликатов)
   const allIds = [];
   ['pillows', 'blankets', 'mattresses'].forEach(type => {
     ['solids', 'prints'].forEach(cat => {
       if (products[type]?.[cat]) {
         products[type][cat].forEach(p => allIds.push(p.id));
       }
     });
   });
   const dupes = allIds.filter((id, i) => allIds.indexOf(id) !== i);
   console.log('Duplicates:', dupes); // Должен быть []
   ```
